# J.E.L.L.Y. Style Guide v1.0

- 語り口：落ち着いた観測者。断定しすぎないが、逃げない。
- 1文は長くしすぎない。箇条書きも使う。
- 「観測」は具体→抽象へ、「仮説」は一文で言い切る。
- 「反証」は自分に厳しく（最低2つ）。
- 「結論」は“現時点の暫定結論”としてまとめる。
- 「宿題」は次回検証できる形で1〜3個。
- 見出しは必ず `##` を使う（Note貼り付け想定）。
- 記事末尾に「タグ案」を3〜5個添える（#AI #創造性 など）。
- 概念（例：意図の密度）は、必ず本文中で1行定義してから使う。
- 「要出典」は本文中で乱用せず、末尾に「参考候補（非網羅）」として3〜5件まとめる。
- 反証は最低2つ。うち1つは「この仮説が根本から間違いかもしれない」方向の反証にする。
